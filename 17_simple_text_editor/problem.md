# Simple Text Editor

## Problem Statement
Implement a simple console-based text editor that allows users to write, read, and save text files.

## Requirements
- Create a new text file or open an existing file.
- Write multiple lines of text.
- Save content to the file.
- Display file content on request.

## Concepts Used
- File I/O
- Loops
- Strings
- Exception handling

## Supported Languages
- Java
- Python
- C++
