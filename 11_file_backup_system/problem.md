# File Backup System

## Problem Statement
Create a program that backs up files from a source directory to a backup directory.

## Requirements
- If the backup directory does not exist, create it.
- Copy only regular files (ignore subdirectories).
- If a file already exists in the backup directory, overwrite it.
- Handle missing source directory gracefully.

## Input
- Source directory path
- Backup directory path

## Output
- Console messages indicating which files were backed up.

## Concepts Used
- File I/O
- Directory handling
- Exception handling

## Supported Languages
- Java
- Python
- C++
