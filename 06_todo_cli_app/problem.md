# TODO CLI Application

## Problem Statement
Create a command-line based TODO application that allows users to
manage daily tasks efficiently.

## Requirements
- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Exit the application gracefully

## Constraints
- Tasks are stored in memory
- No external database or file storage

## Concepts Used
- Lists / Vectors
- Loops
- User input handling
- Conditional logic

## Supported Languages
- Python
- Java
- C++
