# Duplicate File Finder

## Description
This program scans a directory and identifies duplicate files based on
their content using hashing. Files with identical hashes are considered duplicates.

## Features
- Recursive directory scanning
- Content-based comparison (not filename-based)
- Efficient hashing approach
- Console-friendly output

## Languages
- Python
- Java
- C++

## Concepts Used
- File I/O
- Hashing
- Maps / Dictionaries
- Directory traversal

- Detect redundant files in PCs, servers, or cloud storage
