# Login Attempt Tracker

## Problem Statement
Design a login system that tracks failed login attempts and locks the account after a maximum number of failures.

## Requirements
- Allow a maximum of 3 incorrect password attempts.
- If the correct password is entered, login succeeds.
- After 3 failed attempts, lock the account and stop further input.

## Input
- Password entered by the user

## Output
- Login success message OR
- Account locked message

## Constraints
- Password is predefined in the program.

## Concepts Used
- Loops
- Conditional statements
- User input handling

## Supported Languages
- Java
- Python
- C++
