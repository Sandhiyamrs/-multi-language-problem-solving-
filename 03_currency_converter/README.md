# Currency Converter

## Description
A console-based currency converter that converts amounts between
different currencies using predefined exchange rates.

This project focuses on clean logic, input validation, and
maintainable code structure.

## Features
- Supports multiple currencies
- Validates user input
- Clear conversion output
- Easy to extend with new currencies

## Limitations
- Uses static exchange rates (no live API)

## Languages
- Python
- Java
- C++

## Concepts Used
- Maps / Dictionaries
- Arithmetic operations
- User input validation
- Conditional logic
