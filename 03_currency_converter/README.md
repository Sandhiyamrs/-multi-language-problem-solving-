# Currency Converter

## Problem
Convert an amount from one currency to another using live exchange rates.  
Features:
- Supports multiple currencies
- Fetch live rates (API for Python)
- Input validation and error handling

## Input
- Amount
- Source currency (USD, EUR, INR, etc.)
- Target currency

## Output
- Converted amount with 2 decimal points

## Constraints
- Amount > 0
- Supported currencies: USD, EUR, INR, GBP, JPY, AUD, CAD

## Example
Input: 100 USD to EUR
Output: 92.50 EUR

## Why This Problem Is Useful
- Real-world finance applications
- Useful for travelers, e-commerce, and fintech apps
