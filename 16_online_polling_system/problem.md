# Online Polling System

## Problem Statement
Create a console-based polling system where users can vote for multiple options.

## Requirements
- Display a list of candidates/options.
- Accept votes from multiple users.
- Tally votes and display the winner.
- Handle invalid input gracefully.

## Input
- Candidate number
- Votes from users

## Output
- Vote tally
- Winner announcement

## Concepts Used
- Arrays / Lists / Maps
- Loops
- Conditional statements
- Input validation

## Supported Languages
- Java
- Python
- C++
